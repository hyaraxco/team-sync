# Business Model — Team Sync SaaS

> Pricing tiers, maintenance model, dan feature gating.

---

## Pricing Tiers

| Fitur | Free | Pro Monthly | Pro Yearly |
|-------|------|-------------|------------|
| **Harga** | Rp 0 | Rp 199.000/bulan + Rp 5.000/karyawan | Rp 1.990.000/tahun + Rp 50.000/karyawan |
| **Hosting** | Cloud only | Cloud atau Self-hosted | Cloud atau Self-hosted |
| **Domain** | Subdomain only | Subdomain atau Custom | Subdomain atau Custom |
| Max Karyawan | 25 | 200 | 200 |
| Max Teams | 1 | Unlimited | Unlimited |
| Free Trial Pro | 14 hari (tanpa kartu) | — | — |
| Staff Management | ✅ | ✅ | ✅ |
| Attendance (Basic) | ✅ | ✅ | ✅ |
| Teams | ✅ (max 1) | ✅ | ✅ |
| Leave Management | ✅ (basic) | ✅ (full) | ✅ (full) |
| Analytics | ✅ (basic) | ✅ (full) | ✅ (full) |
| Payroll + THR | ❌ | ✅ | ✅ |
| Performance Review + TOPSIS | ❌ | ✅ | ✅ |
| Projects & Tasks | ❌ | ✅ | ✅ |
| Overtime Management | ❌ | ✅ | ✅ |
| Meetings | ❌ | ✅ | ✅ |
| Export Excel/PDF | ❌ | ✅ | ✅ |
| Custom Domain | ❌ | ✅ | ✅ |
| Maintenance | — | Included | Included |
| Support | Community | Priority | Priority |

> **Rationale hybrid pricing**: Model per-employee lebih fair untuk SME Indonesia. Base price rendah menurunkan barrier to entry, biaya scale sesuai jumlah karyawan. Competitor (Talenta, Gadjian, KaryaOne) juga pakai per-employee pricing.

> **Rationale Free tier expansion**: Max 25 karyawan + Leave + basic Analytics supaya user experience lebih banyak value sebelum convert. Free tier = funnel terbaik.

## Planned (Year 2) — Lifetime Tier

| Fitur | Lifetime |
|-------|----------|
| **Harga** | Rp 14.999.000 |
| **Hosting** | Self-hosted only |
| **Domain** | Custom only |
| Max Karyawan | Unlimited |
| Maintenance | 1 tahun included, opsional setelahnya (Rp 1.999.000/tahun) |
| Support | 1 tahun (opsional lanjut) |

> **Deferred**: Lifetime tier ditunda sampai brand trust established dan support costs understood. Launch Year 2 setelah ada 30+ paying customers dan data support cost real.

---

## Maintenance Model (Lifetime — Year 2)

- **Tahun 1**: INCLUDED di harga (support + update)
- **Tahun 2+**: OPTIONAL — Rp 1.999.000/tahun
- Tanpa maintenance: aplikasi tetap jalan, tapi tidak dapat update/support

---

## Refund Policy

| Scenario | Refund |
|----------|--------|
| Pro Monthly: hari 1-7 | Full refund |
| Pro Monthly: hari 8-30 | Pro-rata |
| Pro Monthly: setelah 30 hari | Tidak ada |
| Pro Yearly: bulan 1 | Full refund (minus admin) |
| Pro Yearly: bulan 2-6 | Pro-rata |
| Pro Yearly: setelah bulan 6 | Tidak ada |

## Churn Prevention

| Trigger | Action |
|---------|--------|
| Trial H-3/H-1 | Reminder email + in-app |
| Trial expired | Downgrade ke Free |
| Pro expired H-7 | Reminder perpanjang |
| Pro expired | Grace period 7 hari |
| User tidak aktif 30 hari | "We miss you" email |
| Win-back (30 hari post-cancel) | Diskon 20% |
