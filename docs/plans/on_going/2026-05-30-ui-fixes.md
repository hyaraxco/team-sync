## 1. Notification Badge Standardization

### File Location
`team-sync-fe/src/components/admin/Header.vue`

### Current State
- Staff: Blue badge (`bg-brand-primary`)
- Other roles: Red badge (`bg-danger-500`)

### Target State (All Roles)
- Position: `-right-1 -top-1`
- Shape: **Round** (same width/height)
- Color: `bg-danger-500` (red)
- Size: `h-5 w-5` (or `w-6 px-1` if count > 9)
- Text: `text-[10px] font-bold`

### Changes

Replace:
```javascript
const unreadBadgeClass = computed(() => {
    if (isEmployeeUser.value) {
        const safeCount = Number.isFinite(unreadNotificationCount.value)
            ? Math.max(0, Math.floor(unreadNotificationCount.value))
            : 0;
        const sizeClass = safeCount > 9 ? "w-6 px-1" : "w-5";

        return `absolute -right-1 -top-1 flex h-5 ${sizeClass} items-center justify-center rounded-full border-2 border-white bg-brand-primary text-[10px] font-bold leading-none text-white`;
    }

    return "absolute right-0 top-0 flex h-5 min-w-[1.75rem] -translate-y-1/2 translate-x-1/3 items-center justify-center rounded-full border-2 border-white bg-danger-500 px-1.5 text-[11px] font-bold leading-none tracking-tight text-white";
});
```

With:
```javascript
const unreadBadgeClass = computed(() => {
    const safeCount = Number.isFinite(unreadNotificationCount.value)
        ? Math.max(0, Math.floor(unreadNotificationCount.value))
        : 0;
    const sizeClass = safeCount > 9 ? "w-6 px-1" : "w-5";

    // Standardized red badge for all roles
    return `absolute -right-1 -top-1 flex h-5 ${sizeClass} items-center justify-center rounded-full border-2 border-white bg-danger-500 text-[10px] font-bold leading-none text-white`;
});
```

## 2. ApexCharts Error Fix

### File Location
`team-sync-fe/src/views/admin/analytics/AnalyticsDashboard.vue`

### Problem
Chart tries to render before DOM element is ready, causing "Element not found" error.

### Changes

**Add chartReady ref:**
```javascript
import { ref, onMounted, nextTick } from 'vue';

const chartReady = ref(false);

onMounted(async () => {
    await nextTick();
    chartReady.value = true;
});
```

**Update all VueApexCharts components:**
```vue
<VueApexCharts
    v-if="chartReady && attendanceDeductionSeries[0]?.data?.length"
    type="line"
    height="320"
    :options="attendanceDeductionOptions"
    :series="attendanceDeductionSeries"
/>

<VueApexCharts
    v-if="chartReady && hrCostSeries[0]?.data?.length"
    type="area"
    height="320"
    :options="hrCostOptions"
    :series="hrCostSeries"
/>

<VueApexCharts
    v-if="chartReady && teamPerformanceSeries[0]?.data?.length"
    type="bar"
    height="320"
    :options="teamPerformanceOptions"
    :series="teamPerformanceSeries"
/>
```
