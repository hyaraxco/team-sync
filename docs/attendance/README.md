# Attendance Docs

## Ringkasan Domain
- Domain ini menampung dokumentasi attendance admin dan employee.
- Saat ini attendance sudah terhubung langsung ke payroll melalui:
  - attendance records sebagai source data payroll
  - cut-off readiness untuk generate payroll
  - employee workspace `My Attendance`
